
### interface

In sbt-datatype 0.2.0 the `protocol` type, which generates abstract class, is renamed to `interface`.
It supports `methods` list as well as `fields`. [#14][14] by [@Duhemm][@Duhemm]/[#19][19] by [@eed3si9n][@eed3si9n]
