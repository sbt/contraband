### extra

sbt-datatype 0.2.3 lets you put in extra code on top of the generated code.

    {
      "type": "record",
      "namespace": "com.example",
      "target": "Scala",
      "name": "Person",
      "fields": [
        {
          "name": "name",
          "type": "String"
        }
      ],
      "extra": [
        "def toUpperCase: Person =",
        "  Person(name.toUpperCase)"
      ]
    }

[#31][31] by [@Duhemm][@Duhemm]
